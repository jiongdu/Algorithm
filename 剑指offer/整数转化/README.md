题目描述:
编写一个函数，确定需要改变几个位，才能将整数A转变成整数B。
给定两个整数int A，int B。请返回需要改变的数位个数。
测试样例：
10,5
返回：4

题目解析:
比较简单，统计A转变成B需要改变的数位个数，即统计A与B异或结果中1的个数。
